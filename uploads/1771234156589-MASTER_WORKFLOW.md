# HabitFuel Content Production System
## Complete Workflow for Outsourcing

### Overview
This system produces psychology-focused, habit formation content for students and young adults (16-28) in the style of UnordinaryMind.

**Channel Focus:** Behavioral psychology, neuroscience, habit formation
**Visual Style:** Minimalist hand-drawn sketch animations
**Tone:** Calm, reflective, research-backed (not academic)
**Target:** Students and young adults struggling with habits, procrastination, focus, consistency

## Outsourcing Tips

### For Script Writers:
- Share `02_script_writing.md` prompt
- Provide 1-2 example scripts first
- Request revisions if tone is too academic or motivational
- Check that scientific claims are accurate (even if not cited)

### For Image Generators:
- Share `04_image_generation.md` with the full script
- Provide reference images from UnordinaryMind or similar channels
- Ensure consistency in style across all images
- Request 2 images per scene initially to verify style

### For SEO Specialists:
- Share `06_seo_optimization.md` with completed script
- Emphasize natural language over keyword stuffing
- Check that titles match the calm, curious tone of the channel
